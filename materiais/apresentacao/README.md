# Apresentação — Cursor para PMs

Apresentação em HTML para o curso presencial na Product Arena.

## Arquivo

Use **`index-slides.html`** — slides em tela cheia 16:9, sem sidebar. Abra no navegador para projeção.

## Como usar

1. Abra **`index-slides.html`** no navegador (duplo clique ou `open index-slides.html`) — arquivo oficial para projeção.
2. Navegue com **←** **→** ou **Space**
3. Sem sidebar: slides em tela cheia 16:9

## Estrutura

- Baseada em [Claude-Presos-V2](https://github.com/D-Squared70/Claude-Presos-V2)
- Paleta Product Arena: cinza claro, vermelho #FF5757

## GitHub Pages

Ao publicar o repositório no GitHub Pages, mantenha a pasta **materiais/** como está (com `apresentacao/` e `expectations-analysis/` no mesmo nível). O slide "expectations-analysis" usa um iframe com caminho relativo `../expectations-analysis/index.html`, que funciona quando a URL da apresentação é `.../materiais/apresentacao/index-slides.html`.

## Rule

O prompt que gera apresentações como esta está em `.cursor/rules/html-slides-presentation.mdc`. Mostre para os participantes como exemplo de Rules/prompts que geram output estruturado.
