# Exercício 13: Projeto Final - Parte 5 (Review e Refinamento)

**Objetivo:** Usar a IA como par para revisar código e fazer ajustes finos visuais.

---

## 📝 Passo a Passo

1.  Selecione um arquivo de código importante (ex: `App.jsx` ou um componente).
2.  Use o atalho **`Cmd + Shift + L`** para adicionar o arquivo ao chat (ou arraste o arquivo para a janela do Chat).
3.  Prompt de Review:
    ```text
    "Faça um code review deste arquivo. Há bugs de segurança ou performance? O código está limpo?"
    ```
4.  Prompt de Ajuste Visual (se o preview estiver rodando):
    ```text
    "Aumente o espaçamento entre os cartões de votação e mude a cor do botão de 'Votar' para um verde mais vibrante."
    ```

---

## ❓ Dúvidas e Erros Comuns

**O Cursor mudou algo que eu não queria**
Use o `Cmd + Z` no arquivo para desfazer, ou clique em "Reject" na sugestão do Cursor para descartar a alteração.

**Review muito pedante**
Você pode pedir: "Foque apenas em erros críticos, ignore estilo por enquanto."
