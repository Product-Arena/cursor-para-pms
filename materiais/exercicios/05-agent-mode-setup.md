# Exercício 05: Garantir acesso aos recursos do curso (Modo Agent)

**Objetivo:** Deixar o Cursor agir no seu sistema para garantir que você tenha acesso aos Rules, Commands e recursos do repositório do curso.

---

## 📝 Passo a Passo

1.  Utilize o **Chat (`Cmd + L`)**. Não é necessário ativar nenhum modo especial, apenas autorize os comandos quando solicitado.
2.  Com o contexto da pasta do repositório do curso (`cursor-para-pms`) aberta ou referenciada, digite algo como:
    ```text
    "Quero garantir que eu tenha acesso aos Rules, Commands e recursos do curso neste workspace. Ajuda-me a configurar."
    ```
3.  O Cursor pode sugerir copiar ou vincular arquivos de `.cursor` ou orientar a abrir a pasta do curso como workspace.
4.  **Aprove** a execução dos comandos quando solicitado (se fizer sentido e parecer seguro).

---

## ❓ Dúvidas e Erros Comuns

**Ele perguntou se pode rodar um comando. Devo aceitar?**
Sim, para este exercício. Sempre leia o comando antes (ex: `cp -r ...`). Se parecer seguro, clique em "Run" ou "Allow".

**Nada aconteceu**
Às vezes é preciso abrir a pasta `cursor-para-pms` como workspace (File → Open Folder) para que o `.cursor` da raiz do curso fique ativo. Você também pode copiar o que precisar de `stack/Lucas` ou `stack/Arthur` para sua própria `.cursor`.
