# Exercício 09: Projeto Final - Parte 1 (Setup)

**Objetivo:** Iniciar o projeto final (Sistema de Votação) criando a estrutura base.

---

## 📝 Passo a Passo

1.  Crie uma nova pasta vazia no seu computador chamada `voting-system`.
2.  Abra essa pasta no Cursor (`File > Open Folder`).
3.  Abra o Chat (`Cmd + L`) e digite:
    ```text
    "Quero criar um sistema de votação simples (voting system) para ideias de produtos. Vamos começar pelo PRD. Use o comando /create-prd (se tiver skill) ou crie um documento markdown `docs/PRD.md`."
    ```
4.  O objetivo aqui é ter a estrutura inicial de pastas e o documento de requisitos.

---

## ❓ Dúvidas e Erros Comuns

**Não tenho a skill /create-prd**
Sem problemas. Apenas peça: "Crie um arquivo `PRD.md` descrevendo um sistema onde usuários podem submeter ideias e votar nelas (upvote/downvote)."
