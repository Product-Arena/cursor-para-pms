# Exercício 08: Projeto Final - Parte 1 (Setup)

**Objetivo:** Iniciar o projeto Arena Voting System usando o command `/start-a-business`, que cria a estrutura de pastas e o documento de Product Strategy.

---

## 📝 Passo a Passo

1.  Abra o Chat (`Cmd + L`).
2.  Digite `/start-a-business` e envie.
3.  Quando o agente perguntar, informe o case Arena Voting System:
    - **Nome:** `arena-voting-system`
    - **Descrição:** Sistema de votação de ideias para priorizar melhorias. Usuários votam em propostas e o time acompanha o ranking.
    - **Audiência:** Produtores, PMs e stakeholders que participam de sessões de priorização e precisam decidir o que construir em seguida.
4.  Siga o fluxo guiado — o Business Consultant Agent fará perguntas socráticas para criar a Product Strategy.
5.  Ao final, você terá a pasta `projects/arena-voting-system/` com `documents/` e o arquivo `product-strategy-arena-voting-system.md` prontos para o próximo passo (`/create-prd`).

---

## ❓ Dúvidas e Erros Comuns

**O command /start-a-business não aparece**
Certifique-se de que o repositório do curso está aberto como workspace — os commands ficam em `.cursor/commands/`.

**Quero usar outro nome para o projeto**
O nome da pasta segue o que você informar (ex.: `arena-voting-system`). Use kebab-case.
