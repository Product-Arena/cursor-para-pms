# Exercício 11: Projeto Final - Parte 3 (Design Frontend)

**Objetivo:** Gerar a interface do usuário a partir das especificações.

---

## 📝 Passo a Passo

1.  Use o **Chat (`Cmd + L`)**.
2.  Prompt:
    ```text
    "Com base no PRD.md, crie a página inicial do sistema de votação. Use React, Tailwind CSS e componentes shadcn/ui. Quero um visual limpo e moderno."
    ```
3.  O Cursor começará a criar os arquivos (`App.jsx`, `components/...`).
4.  Se ele pedir para instalar pacotes, permita.
5.  Para visualizar, abra o terminal (`Ctrl + ~`) e rode o comando de dev (geralmente `npm run dev`), depois abra o link `localhost` no navegador ou na aba "Preview" do Cursor.

---

## ❓ Dúvidas e Erros Comuns

**O site está em branco**
Verifique o console do navegador (F12) para erros.
Copie o erro e cole no chat do Cursor: "Deu esse erro, conserte."

**Estilo "quebrado"**
Às vezes o Tailwind não é configurado corretamente de primeira. Peça: "O estilo não aplicou. Verifique a configuração do tailwind.config.js e index.css".
