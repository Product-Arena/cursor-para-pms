# Exercício 10: Projeto Final - Parte 2 (PRD)

**Objetivo:** Refinar os requisitos do produto com ajuda da IA.

---

## 📝 Passo a Passo

1.  Com o arquivo `PRD.md` aberto (ou no contexto), refine as regras de negócio.
2.  Prompt:
    ```text
    "Adicione uma regra de negócio no PRD: um usuário só pode votar uma vez por ideia. Atualize o arquivo."
    ```
3.  Peça para ele sugerir tecnologias:
    ```text
    "Sugira uma stack técnica simples para esse MVP (ex: React + Vite + Supabase ou Firebase). Atualize o PRD com a seção de Arquitetura."
    ```

---

## ❓ Dúvidas e Erros Comuns

**Ele só conversa, não edita o arquivo**
Peça explicitamente no Chat para ele escrever no arquivo e, quando ele sugerir a alteração, clique em "Apply" para aceitar.
