# Questionário Pré-Curso — Cursor para PMs

Questionário para enviar aos participantes antes do curso e entender o perfil, contexto e expectativas de cada um.

---

## Instruções de uso

- Enviar por formulário (Google Forms, Typeform, etc.) ou e-mail
- Prazo sugerido: 1–2 semanas antes do curso
- Respostas ajudam a calibrar exemplos, ritmo e prioridades do dia

---

## Perguntas sugeridas

### 1. Conhecimento atual com Cursor e ferramentas similares

*Como você avalia seu conhecimento hoje?*

- [ ] Nunca usei Cursor
- [ ] Usei pouco (1–2 vezes)
- [ ] Uso ocasionalmente (1–2x por semana)
- [ ] Uso com frequência (quase todo dia)
- [ ] Uso Cursor e outras ferramentas com IA para código (GitHub Copilot, ChatGPT com código, etc.)

*Resposta aberta opcional:* Se já usa Cursor ou similar, descreva em uma frase o que costuma fazer.

---

### 2. Stack de ferramentas no trabalho

*Quais ferramentas você usa no dia a dia? (marque todas que se aplicam)*

- [ ] Google Workspace (Docs, Sheets, Drive, Gmail, Calendar)
- [ ] Jira
- [ ] Confluence
- [ ] Databricks ou outro data warehouse / SQL
- [ ] Notion
- [ ] Figma
- [ ] Outras: _______________________

*Pergunta de acompanhamento:* Qual dessas você **mais** gostaria de integrar com o Cursor no curso?

---

### 3. Experiência com terminal e Git

*Como está sua familiaridade com terminal e Git?*

- [ ] Nunca usei / não sei o que é
- [ ] Conheço pouco (abrir pasta, alguns comandos)
- [ ] Uso básico (git status, add, commit, push)
- [ ] Uso com confiança (branches, merge, pull)

---

### 4. Principal expectativa com o curso

*O que você mais espera aprender ou conseguir fazer após o curso? (escolha até 2)*

- [ ] Entender o que o Cursor faz e como funciona
- [ ] Automatizar tarefas repetitivas (docs, user stories, relatórios)
- [ ] Integrar com minhas ferramentas (Jira, Google, Databricks, etc.)
- [ ] Analisar dados ou codebase sem depender tanto de dev
- [ ] Criar protótipos ou diagramas
- [ ] Outro: _______________________

*Resposta aberta opcional:* Em uma frase, qual seria o sucesso do curso para você?

---

### 5. Caso de uso prioritário

*Imagine que o Cursor já está integrado ao seu fluxo. Qual tarefa você gostaria de fazer primeiro?*

*(resposta aberta, 1–2 frases)*

Exemplos: "Criar user stories no Jira a partir de um PRD", "Analisar funil de ativação no Databricks e gerar insights", "Manter documentação do produto no Confluence atualizada".

---
