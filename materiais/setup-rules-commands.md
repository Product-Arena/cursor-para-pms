# Rules e Commands do Curso

As rules e commands do curso Cursor para PMs estão na pasta **`.cursor/`** na raiz do repositório. Ao clonar o repo e abrir a pasta como workspace no Cursor, elas ficam automaticamente disponíveis.

## Estrutura

| Pasta | Conteúdo |
|-------|----------|
| **`.cursor/rules/`** | Regras de estilo e prompts (PRD, user story para Jira, slides, design system, etc.) |
| **`.cursor/commands/`** | Commands como `/create-prd`, `/start-a-business`, `/new-feature`, etc. |
| **`.cursor/templates/`** | Templates de Product Strategy, PRD, Design System, Marketing Guide |

## Como configurar

1. Clone o repositório do curso
2. Abra a pasta clonada como **workspace** no Cursor (File > Open Folder)
3. As rules e commands passam a estar disponíveis nesse projeto

Não é preciso copiar de outro lugar. Veja a [Etapa 0 — Setup do curso](exercicios/00-setup-curso-ordem-recomendada.md) para o fluxo completo de configuração.
